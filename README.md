# my-markdown-parser
invent own my markdown with simple common markdown syntax

# features:
 [✅] header:
 ```markdown 
 ### header
 ```  
 [✅] text: 
 ```markdown 
   balabala +1s
 ``` 
 [✅] link: 
 ```markdown 
 [前端学习资源](http://t66y.com)
 ``` 
 [✅] header_link:
 ```markdown 
 ### [前端学习资源](http://t66y.com)
 ```
